
Uwezo Pamoja Youths Association - Website Package
-------------------------------------------------
Files included:
- index.html : One-page website (edit and replace images as needed)
- images/ : folder with placeholder image files (hero.jpg, logo.png, history.jpg, gallery1.jpg, gallery2.jpg)

Instructions:
1. Replace the placeholder images in 'images/' with your real pictures:
   - logo.png  -> your association logo (recommended PNG)
   - history.jpg -> image containing history/background text
   - hero.jpg -> main banner/background photo
   - gallery1.jpg, gallery2.jpg -> activity photos
2. Open index.html and replace the Google Form link in the "Join Now" button with your form link.
3. Upload files to GitHub repo root (index.html and images folder) and enable GitHub Pages on the main branch (/ root).
4. Site will be live at https://YOUR-USERNAME.github.io/REPO-NAME
